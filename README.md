# rinatyan
ホーム、支払管理機能、やりたいことリスト機能、スタンプカード機能がある。

###ホーム
ダッシュボード形式
・両者の未払い金の差額を出して、どちらが何円払うべきなのかを表示する
・やりたいことリスト内の日付が2か月以内の日付が近いイベントを表示する
・スタンプシートが当日分押せていなく、期限が近付いてきた場合お知らせが表示される

###支払管理機能
・両者の未支払金の履歴を閲覧、追加、削除ができる。
・支払日から3か月経過したデータは自動的に削除される。
・それぞれのデータには支払金、ステータス、発生日時、支払日時、メモ、支払者のデータがある
・ステータスは未支払、支払済み
・右上のガソリン代計算ボタンからガソリン代計算機能の画面に遷移する
######ガソリン代計算機能
・走行距離(km)と今日のガソリン価格(円/1L)を入力することで今回のガソリン代を算出する
・走行距離÷15=使用ガソリン量(L)
・使用ガソリン量×今日のガソリン価格=今回のガソリン代(円)
・ガソリン代を算出した後請求ボタンを押すことにより支払管理機能の未払いに追加される

###やりたいことリスト機能
・互いのやりたいことを追加、削除、閲覧ができる。
・実行日から3年経過したデータは自動的に削除される
・それぞれのデータにはやりたいこと、ステータス、登録日時、実行日付、イベント期間/日付、マップ、URL、画像、登録者、タグ、やりたい度合いメーター、メモのデータがある
・URLは右端のプラスボタンを押すことで複数記入できる
・タグも右端のプラスボタンを押すことで複数追加できる
・タグはユーザーが作成することができ、途中まで打ち込むことで候補がドロップダウンで出てくる（例：ドライブ、町、☆、☆☆、☆☆☆、星はタスクの重みづけ）
・やりたい度合いメーターは一人ひとり5段階のスライドバーで設定できる。
・イベント期間/日付はデフォルトではイベント日付（1日）であり、右端のプラスボタンを押すことでイベント期間になり開始日付と終了日付の入力欄に変わる。
・ステータスは未実行、実行済み
・実行率を円グラフ、両者の総登録数の割合の円グラフで表示
・ビンゴボタンからビンゴ機能の画面に画面遷移する
######ビンゴ機能
・5*5のビンゴカード
・春夏秋冬でシーズンごとに一枚
・春（3/1~5/31）、夏(6/1~8/31)、秋（9/1~11/30）、冬（12/1~2/28）
・ビンゴカードの両脇にある＜または＞ボタンを押すことで前後のシーズンに移動して閲覧できる。
・シーズン終了後はマスクリア条件を満たしてもクリア判定されず、シーズン終了済みのビンゴカードの編集ボタンもなくなり閲覧のみとなる。
・ビンゴカードはシーズン開始で自動生成される。
・ビンゴカード生成から3年経過すると自動削除される。
・通常モードでは閲覧のみが可能となっており、マスを押すことで詳細を閲覧することができる。
・編集ボタンから編集モードになり、マスを押してやりたいことリストから引用することができる。
・編集の際、ビンゴ期間内の日付か日付未設定かつ未実行かつ編集中のビンゴカードに入っていないタスクのみ候補として表示される
・実行済みになることでビンゴのマスクリアとなり、そのマスがクリア状態の見た目になる
・タテヨコナナメいずれか５マスクリアしビンゴになると報酬を得ることができる。
・ビンゴカードの下にビンゴの報酬について閲覧・編集できるものがある。

###スタンプカード機能
・1人1個の毎日実行する目標を定め、クリアできたと感じたらスタンプを押す
・30個マスがあるスタンプカード
・スタンプカードはカードが埋まり、次のカードが生成された後スタンプ日時などのデータは削除され、HTMLのみの保存となる。
・スタンプカードの両脇にある＜または＞ボタンを押すことで前後のカードに移動して閲覧できる
・スタンプは1日1個であり、1日の期限は00:01~24:00である。
・30個スタンプされカードが埋まったら報酬を得ることができる。
・スタンプカードが埋まった後は次のスタンプカードが自動的に生成される。
・スタンプカードの下にスタンプの報酬について閲覧・編集できるものがある。

###マイページ機能
・アカウント情報の確認
・名前、パスワードの変更
・テーマカラー、アイコンの変更
・テーマカラーはカラーピッカーで自由に色を作ることもできる
・アイコンは画像を追加することもできる
・ログアウトができる
